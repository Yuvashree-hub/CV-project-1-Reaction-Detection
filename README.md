# Real-Time Smile Detector

## Description
Detects smiles in real-time using a webcam. Faces are highlighted with green if smiling and red if not. Uses OpenCV Haar cascades for face and smile detection. Lightweight, real-time, and ideal for showcasing computer vision skills.

## Features
- Real-time face detection
- Real-time smile detection
- Color-coded feedback (green = smiling, red = not smiling)
- Works on any webcam
- Lightweight and easy to run

## Requirements
- Python 3.x
- OpenCV
